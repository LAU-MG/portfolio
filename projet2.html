Contexte

Le besoin :

L’environnement technologique :

Le site web était codé en .NET pour la partie back-end. La partie front-end s’est faite sur le CMS en HTML et CSS, pour certaine page j’utilisait les outils du CMS qui générait des éléments dans la page. La base de donnée était en PHP.
